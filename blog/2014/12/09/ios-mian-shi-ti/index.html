Run Loop

Run Loop 是一个循环，你的线程进入并使用它来运行响应输入事件的事件处理程序。你需要提供实现循环部分的控制语句，就是要有while for语句驱动 Run Loop。 在你的循环中，是哟高 run loop object来运行事件处理代码，它响应接收到的事件并启动已经安装的处理程序。
Run Loop 接受两个来源的输入事件：输入源 和 定时源。 输入源传递异步事件，通常消息来自于其他线程或程序。 定时源则传递同步事件，发生在特定事件或者重复的事件间隔。两种输入源都使用一个特定的程序处理例程来处理到达的事件。
